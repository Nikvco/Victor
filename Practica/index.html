SELECTORES CSS

Los selectores CSS son herramientas utilizadas para definir el estilo que quieres dar a tus elementos en CSS. Existen muchos tipos de selectores en este lenguaje, cada uno con su propia sintaxis y utilidad. El uso de las reglas de programación correctas ayuda al explorador a aplicar propiedades específicas a los elementos indicados.

  • Selector
    • El elemento HTML en el que comienza la regla. Esta selecciona el(los) elemento(s) a dar estilo (en este caso, los elementos < p > ). Para dar estilo a un elemento diferente, solo cambia el selector.
  • Declaración
    • Una sola regla como color: red; especifica a cuál de las propiedades del elemento quieres dar estilo.
  • Propiedades
    • Maneras en las cuales puedes dar estilo a un elemento HTML. (En este caso, color es una propiedad del elemento < p > ). En CSS, seleccionas qué propiedad quieres afectar en tu regla.
  • Valor de la propiedad
    • A la derecha de la propiedad, después de los dos puntos (:), tienes el valor de la propiedad, para elegir una de las muchas posibles apariencias para una propiedad determinada (hay muchos valores para color además de red).

Nota las otras partes importantes de la sintaxis:
  1. Cada una de las reglas (aparte del selector) deben estar encapsuladas entre llaves ({}).
  2. Dentro de cada declaración, debes usar los dos puntos (:) para separar la propiedad de su valor.
  3. Dentro de cada regla, debes usar el punto y coma (;) para separar una declaración de la siguiente.
De este modo para modificar varios valores de propiedad a la vez, solo necesitas escribirlos separados por punto y coma (;), así:

p {
col color: red;
width: 500px;
border: 1px solid black;
}

Tipos de selectores CSS

  1. Selector universal
  2. Selector de clase
  3. Selector de ID

  1. Selector universal

    2. El asterisco (*) es el selector universal en CSS. De forma automática, el asterisco selecciona todos los elementos en un documento.

    *{ color: orange; }

  2. Selector de clase

    3. Los selectores de clase son herramientas que, como su nombre lo indica, permiten seleccionar todos los elementos que tienen un mismo nombre de clase.

    < h1 class="oastoral"> hola< /h >
    .oastoral { color: orange; }

  3. Selector de ID

    4. Un selector de ID está diseñado para seleccionar elementos con base en su atributo de ID. Por ejemplo, #toc seleccionará todos los elementos que tengas como ID «toc».

    < h1 id="under">Hola < /h1 >
    #under{ color: orange; }

Hoja de estilo CSS Externa

La manera de vincular estas reglas a una página HTML consiste en referencia el fichero .css desde esa página. De este modo se puede crear un fichero global para una web entera y usarlo en todas las páginas que se quiera.

< link rel="stylesheet" type="text/css" href="/css/estilos-1.css" />
< link rel="stylesheet" type="text/css" href="/css/estilos-2.css" />

  1. Maquetacion

    • width:

      • Ancho de un elemento.

      div { width: 350px;}

    • height:

      • Alto de un elemento.

      div { height: 100px; }

    • vertical-align

      • Alineamiento vertical dentro de un elemento.

      div { vertical-align;}

    • margin:

      • Espacio que se añade entre el elemento y sus vecinos. Se puede diferencia por lado (arriba, abajo, izquierda, derecha).

      div { margin-top: 10px; margin-right: 10px; margin-bottom: 10px; margin-left: 10px; }

    • padding:

      • Relleno interior que se añade en los bordes del A diferencia de margin, cuenta para el tamaño del elemento.

      div { padding-top: 10px; padding-right: 10px; padding-bottom: 10px; padding-left: 10px; }

    • float:

      • Mueve el elemento todo lo posible hacia el lado indicado. Esta propiedad se usa en el posicionamiento flotante de CSS. El tema del posicionamiento en CSS no es trivial y conviene estudiar cómo funciona antes de usar esta propiedad.

      .img1 { float: left; margin-right: 1em; margin-bottom: 1em; }


  2. Fuentes y texto

    • font-family:

      • Tipo de letra

      p { font-family: Georgia, Cambria, Times, "Times New Roman"; }

    • font-size:

      • Tamaño de letra

      p { font-size: xx-large;}
      p { font-size: x-large;}
      p { font-size: large;}
      p { font-size: medium;}
      p { font-size: small;}
      p { font-size: x-small;}
      p { font-size: xx-small;}

    • font-weight:

      • Peso (normal, negrita, …)

      < p style="font-weight: bold;" >

    • font-style:

      • Estilo (normal, cursiva, …)

      < p style="font-style: italic;" >

    • text-decoration:

      • “Decoraciones” como subrayado, tachado, etc.

      a:{ text-decoration: underline; }

    • text-align

      • Alineación del texto (izquierda, derecha, etc.)

      .imagen1 { text-align: center; } div { text-align: center; }

    • text-transform:

      • Mostrar un texto en mayúsculas, minúsculas o la primera letra de cada palabra en mayúsculas.

      p { text-transform: capitalize; }


  3. Color y fondos

    • color:

      • Color del elemento. Se puede especificar en diferentes formatos como palabras predefinidas (red, green, etc.) RGB o como valor hexadecimal.

      body { color: #CCC; }

    • background-color:

      • Color del fondo del elemento.

      div { background-color: #E0ECFF; }

    • background-image:

      • Permite especificar una imagen de fondo.

      h1, h2, h3 { background-image: url("imagenes/fondo.png"); }

    • background-repeat

      • Permite usar una imagen a modo de mosaico en diferentes modalidades.

      div { background-image: url("images/imagen_pequena.png"); background-repeat: no-repeat; }

    • box-shadow:

      • Crear un efecto de sombra para un elemento.

      #sombraclara{ width: 200px; padding: 10px; background-color: #999; color: #fff; box-shadow: 2px 2px 2px #ffc; -webkit-box-shadow: 2px 2px 2px #ffc; -moz-box-shadow: 2px 2px 2px #ffc; }


  4. Listas

    • list-style-image:

      • Usar la imagen especificada como viñeta para la lista.

      ul#tercera li { list-style-image: url("imagenes/estrella.png"); }

    • list-style-type:

      • Diferentes estilos de viñetas y estilos de numeración para elementos de lista.

      ul li { list-style-type: square; }


  5. Bordes

    • border:

      • Añade un borde a un elemento y establece algunas propiedades (grosor, estilo de línea, etc.)

      div { border-top: 1px solid #C00; border-right: 1px solid #C00; border-bottom: 1px solid #C00; border-left: 1px solid #C00; }

    • border-color:

      • Color del borde.

      div { border-top-color: red; border-right-color: red; border-bottom-color: red; border-left-color: red; }

    • border-style:

      • Diferentes estilos para el borde (sólido, puntos, etc.)

      div { border-top-style: solid; border-right-style: double; border-bottom-style: dashed; border-left-style: double; }

    • border-radius:

      • Permite crear esquinas redondeadas para un elemento.

      #example2 { border: 2px solid red; border-radius: 50px 20px; }
